**E-Commerce Sales Analysis Dashboard**

**Project Overview**
This project analyzes a Brazilian e-commerce dataset to understand sales trends, product performance, regional demand, and delivery efficiency.

**Tools Used**
1. Python (Pandas, NumPy)
2. SQL
3. Power BI

**Dataset**
Brazilian E-commerce Public Dataset (Olist)

**Project Workflow**
1. Data cleaning and preprocessing
2. Feature engineering
3. Exploratory Data Analysis
4. Business insights extraction

**Key Insights**
1. Revenue shows steady growth over time
2. Most orders come from Sao Paulo(SP) state
3. Certain product categories dominate sales
4. Average delivery time is around 23 days
